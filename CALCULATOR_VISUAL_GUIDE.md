## Accessibility Features

### Keyboard Navigation
- Tab through all sliders
- Arrow keys adjust values
- Enter on CTA activates link
- Escape closes tooltips

### Screen Reader
- ARIA labels on all inputs
- Role="slider" on range inputs
- aria-valuenow, aria-valuemin, aria-valuemax
- aria-describedby for tooltips

### Visual Accessibility
- High contrast text (4.5:1 minimum)
- Focus indicators visible
- Color not sole indicator (icons + text)
- Readable font sizes (14px minimum)

## Calculator Types Comparison

### Type 1: Time-Saved (EIN Filing)
**Visual Focus:** Clock icons, time metrics highlighted
**Color Scheme:** Orange primary, green for savings
**Key Metrics:** Hours saved, money value, ROI

### Type 2: Risk-Mitigation (Legal)
**Visual Focus:** Shield icons, protection emphasis
**Color Scheme:** Yellow for costs, green for protection
**Key Metrics:** Lawsuit costs, damages, protection value

### Type 3: Tax-Savings (Accounting)
**Visual Focus:** Dollar signs, percentage savings
**Color Scheme:** Yellow for taxes, green for savings
**Key Metrics:** Current taxes, savings, payback period

### Type 4: ROI (Marketing/Social/SEO)
**Visual Focus:** Trending up arrows, revenue growth
**Color Scheme:** Blue for current, green for improved
**Key Metrics:** Current revenue, projected revenue, ROI

### Type 5: Revenue-Growth (Strategy/Coaching)
**Visual Focus:** Growth charts, future projections
**Color Scheme:** Green for growth, blue for investment
**Key Metrics:** Projected revenue, increase, multiple

### Type 6: Automation-Savings (AI/CRM/IT)
**Visual Focus:** Efficiency icons, time savings
**Color Scheme:** Orange for hours, green for savings
**Key Metrics:** Hours automated, annual savings, payback
